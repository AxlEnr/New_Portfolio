# **App Name**: Axl Portfolio

## Core Features:

- Static Content Display: Display content on a single-page website including personal introduction, tech stack, projects, and certificates.
- Multi-language support: Implement internationalization (i18n) for English and Spanish content.
- Expandable Bio Section: Initial introduction with an option to expand for a more detailed view.
- Project Showcase with Details Dialog: Display projects with images, descriptions, and links to code, with a dialog for detailed views via a carousel.
- Downloadable Resume: Implement language-aware resume downloads (English/Spanish) through a creative navigation button.
- Contact Links Footer: Contact links in the website footer.

## Style Guidelines:

- Primary color: Soft, vibrant blue (#64B5F6) to create a calm, professional, and visually appealing experience.
- Background color: Dark charcoal gray (#333333) to create a sleek, modern, and sophisticated appearance.
- Accent color: Light blue-green (#80CBC4) to provide subtle highlights, indicating interactive elements and important information without overwhelming the dark background.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, providing a clean, readable, and modern feel.
- Simple, line-based icons for representing technologies and project categories.
- Single-page layout with clear section divisions for easy navigation.
- Subtle transitions when opening project details in the dialog.
